O1 ECO

SYNOPSIS

This project introduces N2O protocols for Standard ML programming language. The aim is to be compatible at transport level with Erlang version of N2O. This ecosystem represents a solid CORBA/OASIS/SOAP/XMPP-replacement, ready for high-speed, low-latency IoT, MQTT and WebSocket applications.
